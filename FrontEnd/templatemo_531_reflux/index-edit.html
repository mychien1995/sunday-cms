EpiServer Dynamic Lucene Extension

EpiServer kinda forced everyone to use EpiFind really. And despite costing money, EpiFind has a limit request/sec and the Find cluster is unstable as well. So i wrote this extension for you to index & search your contents in a distributed setup. And i'm using Lucene becaue its free, and it does not require you to have a search server. What i'm doing is to store Lucene index in Azure Blob Storage, you contents will be index and search from the storage. If you don't have an Azure Blob Storage that is ok too. I have support for storing index in file system as well, the index will be synced between nodes using Azure Event. And the best part is you can custom your index field to search using .config file.
